How to edit this document

To edit this document you will need to use git to make a local copy on your computer. Edit changes and than push it back to the server.
Plese remember you need to make changes to "gh-pages" branch to be able to publish it on github.
It's explained below in more details

  1. Create folder on your machine
  2. Clone this repository git clone https://github.com/tahoesasquatch/code-standards .
  3. Checkout gh-pages branch: git checkout gh-pages
  4. Make your changes (using your favorite editor)
  5. Commit your changes git commit -a -m "something smart you just did"
  6. Push your changes to the server git push origin
